# Llama 3 Student Productivity Counselor

This repository contains a fine-tuned **Llama 3 (8B)** model specialized in analyzing student lifestyle habits and providing data-driven productivity advice.

By leveraging **QLoRA** and the **Unsloth** library, the model was trained to interpret digital distractions and sleep patterns to suggest actionable improvements.

---

##  Project Overview

- **Model:** Llama 3 (8B)  
- **Method:** QLoRA Fine-Tuning  
- **Framework:** Unsloth & Hugging Face  
- **Objective:** Specialized AI coaching for student time management  

---

##  Dataset

The dataset used for this project was sourced from Kaggle:

- **Dataset Name:** Student Productivity and Digital Distraction Dataset  
- **Size:** 20,000 samples  
- **Features:**  
  - Study Hours  
  - Sleep Hours  
  - Screen Time  
  - Productivity Score
